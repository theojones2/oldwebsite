How to force a WordPress blog to use https

1 min read

Using the WordPress Force HTTPS plugin, I have configured this blog such that it redirects to https. Its a very simple plugin to use, just install and activate it (from the official WordPress repositories) and your blog will redirect to HTTPS. Make sure you have your https certificates configured properly first. There are a number of reasons to force https whenever possible.
