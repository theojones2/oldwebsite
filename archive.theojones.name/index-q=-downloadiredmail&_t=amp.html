Some Old Notes: Configuring a Email Server with Iredmail and Ubuntu 12.04

1 min read

Install Iredmail

See the Iredmail documentation http://www.iredmail.org:80/install_iredmail_on_ubuntu.html">http://www.iredmail.org/installiredmailon_ubuntu.html for more details

First install bzip2

apt-get install bzip2

Edit first line of /etc/hosts

nano /etc/hosts

To read like the following

127.0.0.1       mail.theojones.name localhost mail

 

Download Ired Mail

wget https://bitbucket.org/zhb/iredmail/downloads/iRedMail-0.8.5.tar.bz2

Untar Ired mail.

tar xjf iRedMail-0.8.5.tar.bz2

Add forwarding

Go to https://mail.theojones.name/phpldapadmin/htdocs/cmd.php
