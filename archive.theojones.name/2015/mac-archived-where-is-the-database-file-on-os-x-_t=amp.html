[mac] (Archived) Where is the database file on OS X and can we change it? - Mac Help - Evernote User Forum

If you downloaded directly it's in user/library/application support/evernote

If you downloaded from the app store it's here ~/Library/Containers/com.evernote.Evernote

And, if you have an iMac with a fusion drive, it might be here

/Library/Containers/com.evernote.Evernote/Data/Library/Application Support/Evernote/account// 
 
It's not supported if you change it, but you can try if you understand aliasing.
